# рџљЂ Installation

Clone the repository

```bash
git clone https://github.com/nensi25/stadium-os-ai.git
```

Go to the project

```bash
cd stadium-os-ai
```

Install backend dependencies

```bash
cd backend
npm install
```

Run backend

```bash
npm start
```

Run backend tests

```bash
npm test
```
